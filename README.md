# Second Hand Market

A console application that simulates a simplified online second-hand marketplace.

Users can register, list items for sale, browse listings, purchase items, and leave reviews.

## Features

- User registration and login
- Create and manage item listings
- Browse and search listings
- Purchase items
- Leave seller reviews
- View transaction history

## Technologies

- C#
- .NET
- Object-Oriented Programming
- LINQ
- Generic Collections
- Git for version control

## How to Run

1. Open the project in JetBrains Rider
2. Build the solution
3. Run the console application
